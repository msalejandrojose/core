# Media de la plantilla María

Suelta aquí tus archivos con **exactamente estos nombres**. La página los
detecta sola (comprobación en tiempo de compilación): si el archivo existe se
usa; si no, se muestra una imagen de marcador. No hay que tocar código.

| Nombre de archivo      | Dónde aparece                                   | Contenido                          |
|------------------------|-------------------------------------------------|------------------------------------|
| `hero.jpg`             | Hero — fondo a pantalla completa                | Huerto de árboles (aéreo)          |
| `hero.mp4` *(opcional)*| Hero — si existe, sustituye a la imagen por vídeo | Uno de tus 3 vídeos              |
| `girasoles.jpg`        | Sección 2 — imagen protagonista                 | Campo de girasoles                 |
| `cereal.jpg`           | Díptico — imagen izquierda                       | Cereal con riego circular          |
| `huerta-detalle.jpg`   | Díptico — imagen derecha                          | Huerta, detalle (sandías/calabazas)|
| `huerta.jpg`           | Sección 4 — imagen a todo el ancho              | Huerta en hileras (aéreo)          |

## Cómo añadirlas

1. Arrastra las imágenes a esta carpeta (`apps/web/public/media/maria/`) y
   renómbralas según la tabla.
2. Guarda / recarga el navegador. Aparecen automáticamente.

> El díptico recorta a formato vertical (4:5); las fotos horizontales se
> adaptan sin problema.

## Vídeo del Hero (opcional)

Si quieres vídeo en lugar de la imagen `hero.jpg`, añade `hero.mp4`:
- Formato `.mp4` (H.264), horizontal, 1920×1080 o superior.
- Sin audio (se reproduce en silencio) y en bucle limpio.
- Peso ideal < 6–8 MB para carga rápida.
- Conserva también `hero.jpg` como primer fotograma / respaldo.
